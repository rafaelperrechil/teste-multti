**Instalação**

```cmd
git clone https://github.com/rafaelperrechil/teste-multti.git
```
Na pasta do projeto:
```cmd
cd api
npm install -g json-server
json-server --watch db.json --port 5000
cd..
cd web
npm install
npm start
```
